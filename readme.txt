Pour pouvoir utiliser le site veuillez suivre le schema ci-dessou:
- ouverture de l'invité de commande
-utiliser l'inviter de commande pour etre dans le dossier backend 
-faite npm install
-creation d'un dossier "images" à la racine du projet pour pouvoir utiliser le site et avoir les images qui  s'affiche).
-creation du fichier .env à la racine du projet avec les bonne données dans les bonne variables.
